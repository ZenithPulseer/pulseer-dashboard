# Design System

Pulseer Dashboard uses a clean, professional monitoring interface built with Tailwind CSS and shadcn/ui.

## Typography

- **Font**: Inter (Sans-serif) for high readability.
- **Monospace**: For addresses, hashes, and JSON payloads.

## Spacing & Layout

- Consistent 8px grid system.
- Responsive sidebar navigation.
- Card-based layouts for summaries and detail views.

## Status Indicators

- **Healthy**: Green (`#22C55E`)
- **Degraded**: Yellow (`#EAB308`)
- **Unhealthy**: Red (`#EF4444`)
- **Info**: Blue (`#3B82F6`)
