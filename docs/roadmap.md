# Roadmap

Pulseer Dashboard is evolving to become a comprehensive monitoring hub.

## Short-term

- [ ] Persistent backend status storage.
- [ ] Real-time Soroban event subscriptions (WebSockets).
- [ ] WalletConnect integration for safe account selection.

## Long-term

- [ ] Multi-network concurrent monitoring.
- [ ] Custom rule builder UI.
- [ ] Incident timeline views.
- [ ] Exportable monitoring reports (PDF/SARIF).
