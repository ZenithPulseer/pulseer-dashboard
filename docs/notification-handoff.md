# Notification Handoff

Pulseer Dashboard prepares alerts for external notification systems.

## Payload Builders

The dashboard generates standardised JSON payloads for different alert types. These payloads include all the necessary context (severity, evidence, remediation) for delivery.

## Redaction

Sensitive information (like precise amounts or internal IDs) can be redacted in the notification preview to ensure data privacy before handoff.

## Integration

To connect to Slack, Discord, or Email:
1.  Configure a webhook or notification sender in your backend.
2.  Use the `NotificationPayload` generated by the dashboard as the message body.
